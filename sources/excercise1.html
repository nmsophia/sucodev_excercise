Tags

HTML consists from marks (so-called tags). By itself it allows to give the elements on the page some meaning and this is it's main task. Firstly HTML was used for graphic styling of pages but the resulting websites were confusing so it has been limited just to web content.

Doctype

The very first tag in HTML document is doctype. It defines that the text file is a HTML document.

HTML
